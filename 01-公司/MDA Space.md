# MDA Space

## 公司简介

MDA Space（MDA Ltd）是加拿大最重要的航天技术公司，成立于 1969 年，总部位于安大略省。公司以 Canadarm 机械臂闻名，业务覆盖卫星制造、地球观测、机器人与在轨服务以及地面站系统。2024 年获得 Telesat Lightspeed 星座制造合同后进入加速增长期。

## 产业链位置

- 核心位置：卫星制造与系统集成、太空机器人
- 延伸位置：[[02-主题/轨道基础设施]]（Canadarm3）、对地观测（RADARSAT）
- 关键角色：国际空间站 Canadarm2 的制造商与运维方
- 国家维度：加拿大航天（非美国体系，独特的国际合作视角）

## 核心业务

- 卫星制造：Telesat Lightspeed LEO 星座、通信/地球观测卫星
- 太空机器人：Canadarm 系列（ISS Canadarm2、Gateway Canadarm3）
- 地球观测：RADARSAT 系列雷达卫星及地面处理系统
- 地面站系统：卫星运控与数据接收站

## 关键产品/服务

- Canadarm3：为 NASA Gateway 月球空间站设计的下一代机械臂
- Telesat Lightspeed 卫星：作为主承包商制造 Telesat 的 LEO 通信星座
- RADARSAT 星座任务（RCM）：全天候/全天时 SAR 对地观测
- MDA AURORA：卫星制造数字化平台

## 与 SpaceX 的关系

- MDA 与 [[01-公司/SpaceX]] 不直接竞争，但制造的卫星会通过 SpaceX 发射。
- Canadarm3 将安装在 Gateway 空间站上，而 Gateway 的部分模块可能由 SpaceX 发射。
- MDA 代表非美国航天企业如何在以 SpaceX 为核心的全球发射市场中定位。

## 研究观察

- MDA 是观察非美国航天公司如何在 SpaceX 主导的市场中保持竞争力的重要样本。
- Telesat Lightspeed 合同是 MDA 从"组件/子系统级"向"整星座主承包商"跃迁的关键转折点。
- 加拿大航天产业高度依赖 MDA，这既是优势（政府锁定合同）也是风险（政策依赖）。
