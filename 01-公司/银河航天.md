# 银河航天

## 公司简介

银河航天是中国商业航天和卫星互联网领域最值得跟踪的公司之一，核心方向是低轨宽带通信卫星、卫星批量制造和天地融合通信网络。它也是观察中国商业卫星互联网从技术验证走向组网与应用落地的关键样本。

## 核心业务

- 低轨宽带通信卫星研制
- 通信载荷、卫星平台与相关核心能力研发
- 批量化卫星制造与星座建设能力验证
- 天地融合通信网络与应用验证

## 关键产品/服务

- 低轨宽带通信卫星
- “小蜘蛛网”低轨宽带通信试验星座
- 面向车载、海上、边远地区等场景的天地融合通信验证能力

## 与 SpaceX 的关系

- 与 [[01-公司/SpaceX]] 的对照重点不在当前规模，而在“卫星互联网是否能形成系统能力与应用落地”的方向。
- 与 [[01-公司/Amazon Project Kuiper]] 相比，银河航天更像中国本土卫星互联网技术路线与制造路线的代表。
- 它与 [[01-公司/蓝箭航天]] 一起，构成中国侧“造星 + 织网 + 运载”的关键观察组合。

## 研究观察

- 银河航天的核心看点是：它不只是发一两颗试验星，而是在推动低轨宽带卫星的批量研制、组网验证和应用落地。
- 中国卫星互联网的发展节奏会受到政策、组网节奏和国家级星座建设影响，因此银河航天既有市场属性，也有很强的产业政策属性。
- 它适合被当作“中国卫星互联网产业真实工程进展”的观察窗口，而不是单纯题材公司。

## 个人观察

- 关注点：银河航天能否持续证明低成本批量造星和应用验证能力，而不只是停留在试验性展示。
- 关注点：在中国大规模卫星互联网建设中，它最终扮演的是独立运营者、供应商，还是关键技术平台公司。

## 待验证问题

- 银河航天未来更大的价值会来自卫星制造能力，还是来自网络运营与解决方案能力？
- 在国家级星座推进的背景下，商业公司能保留多大程度的独立商业空间？
