# Viasat

## 公司简介

Viasat 是美国卫星通信与网络服务的重要参与者，可作为观察传统卫星通信公司如何应对低轨卫星互联网冲击的代表样本。

## 产业链位置

- 核心属于 [[02-主题/卫星通信]]
- 与 [[02-主题/低轨卫星互联网]] 有显著交集
- 位于 [[03-国家地区/美国商业航天]]

## 核心业务

- 卫星宽带与网络服务
- 航空、海事和政府通信服务

## 关键产品/服务

- 卫星通信容量与服务
- 行业通信解决方案

- [[01-公司/Amazon Project Kuiper]] 的推进也会加剧该赛道竞争。
