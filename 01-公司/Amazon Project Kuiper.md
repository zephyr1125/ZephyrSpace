# Amazon Project Kuiper

## 公司简介

Project Kuiper 是亚马逊的低轨卫星互联网项目，现已正式以 Amazon Leo 品牌对外推进。它的目标是建设覆盖全球的低轨宽带网络，为家庭、企业、政府、航空等客户提供连接服务。

## 核心业务

- 建设 3000+ 颗卫星的低轨互联网星座
- 建设网关站、地面网络与全球接入体系
- 研发不同档位的用户终端
- 将卫星网络与亚马逊既有云、渠道和企业客户资源结合

## 关键产品/服务

- Amazon Leo / Project Kuiper 星座
- Leo Nano、Leo Pro、Leo Ultra 用户终端
- 面向家庭、企业、政府、航空场景的宽带连接服务

## 与 SpaceX 的关系

- 与 [[01-公司/SpaceX]] 旗下 Starlink 是当前最直接的美国低轨宽带竞争关系。
- Kuiper/Amazon Leo 的独特性在于母公司具备云、设备、供应链和渠道资源，而 SpaceX 的独特性在于发射与星座部署一体化。
- 对 [[01-公司/Viasat]] 等传统卫通公司而言，Kuiper 和 Starlink 都在抬高行业竞争标准。

## 研究观察

- Kuiper/Amazon Leo 是观察“大型科技平台切入商业航天”的最好样本之一。
- 它是否成功，不只取决于卫星性能，还取决于终端成本、发射节奏、客户签约能力以及亚马逊生态协同。
- 相比 Starlink，Amazon Leo 在部署速度上起步较晚，但在企业合作和渠道整合上有独特潜力。

## 个人观察

- 关注点：亚马逊能否把云、企业销售、设备和卫星网络真正打成闭环。
- 关注点：服务 rollout 的时间和早期客户获取质量，决定这个项目更像“长期基础设施”还是“高投入试验”。
