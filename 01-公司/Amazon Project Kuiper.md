# Amazon Project Kuiper

## 公司简介

Project Kuiper 是亚马逊布局低轨卫星互联网的重要项目，目标是建设与 Starlink 对标的卫星宽带网络。

## 产业链位置

- 属于 [[02-主题/低轨卫星互联网]]
- 与 [[02-主题/卫星通信]] 强相关
- 位于 [[03-国家地区/美国商业航天]]

## 核心业务

- 低轨卫星互联网网络建设
- 终端与接入服务
- 配套云与基础设施协同

## 关键产品/服务

- Kuiper 星座
- 用户终端

## 与 SpaceX 的关系

- 与 [[01-公司/SpaceX]] 在低轨卫星互联网领域直接对标。
- 对 [[01-公司/Viasat]] 等传统卫星通信玩家形成竞争压力。
