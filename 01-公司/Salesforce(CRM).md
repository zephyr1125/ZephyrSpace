# Salesforce (CRM)

> **公司一句话**：Salesforce 是全球最大企业 CRM 软件公司，真正的利润引擎是高粘性的 SaaS 订阅收入（含 Service/Sales Cloud 双核心），而市场当前给予 ~13x Non-GAAP 远期 PE，是在以"成熟低增速软件"而非"AI再加速平台"定价——核心押注是 Agentforce 能否将营收增速从 10% 重新拉升至 15%+，触发估值重估。

## 模块 2：行业地位与护城河

### 市场地位（事实 + 推断）

- 全球 CRM 市场份额：**约 22-23%**（排名第一，来源：IDC/Gartner 2025）
- 客户数：**320,000+** 全球企业客户
- AppExchange 生态：**7,000+** 第三方应用
- 竞争对手：Microsoft Dynamics 365（~5-15% 份额，增速快于 Salesforce）、SAP CRM、Oracle、ServiceNow（功能重叠）

### 护城河评估

| 护城河来源 | 强度 | 说明 |
|---|---|---|
| 转换成本 | ★★★★★ | 企业 CRM 替换需要数据迁移+员工培训+流程重建，平均替换周期 3-5 年 |
| 生态系统 | ★★★★☆ | AppExchange 7000+ 应用形成正向锁定，ISV 深度依赖 Salesforce |
| 数据网络效应 | ★★★☆☆ | Data Cloud 的数据复合护城河逐渐成型，但仍早期 |
| 品牌/销售团队 | ★★★★☆ | 企业销售关系深，"Salesforce 是 CRM 标配"的心智 |

### 关键竞争威胁：Microsoft Dynamics + Copilot

**事实**（来源：CRM Software Insider，2026）：
- Microsoft Dynamics 365 CRM 收入约 $5.45B/年，增速 ~11-12%，高于 Salesforce 的 ~10%
- Microsoft 优势：Office 365/Teams/Azure 全栈整合，Fortune 500 中 60%+ 已用 Copilot
- Salesforce 反击：Agentforce 专注 "自主完成工作任务"（而非 "给建议"），差异化为 AI 原生平台
- **结论（推断）**：Microsoft 正在从中端市场和 "全栈 Microsoft 已购户" 切入，威胁真实但有限——大企业 CRM 核心仍高粘性，短期替换概率低。中期风险需持续监控，特别是 Copilot Studio 的 Agent 能力是直接竞争。

- 增长是有机为主（FY2026 无重大并购入帐），质量较高

## 综合评估

### A. 公司一句话

Salesforce 是全球 CRM 事实标准，基于极强转换成本护城河经营 $41.5B 订阅收入，FY2026 的 FCF 机器属性（FCF Margin 34.7%）已充分成熟；当前市场以 13x Non-GAAP Forward PE 定价，相当于完全无视 Agentforce 期权性——若 Agentforce 能将营收增速从 10% 推至 15%+，估值将显著重估。

### B. 买入逻辑（3 条，可验证）

1. **Agentforce 货币化已启动，且在加速**（事实）：FY2026 Q4 Agentforce ARR $800M，+169% YoY，29,000+ 付费交易。但仍占总收入约 2%，FY2027 全年需验证能否达到 $1.5-2B ARR 量级（即 ~4-5% 营收占比），才能实质性推动总增速。
2. **估值处于历史极低分位，回购提供强大支撑**（事实）：$50B 回购约占市值 30%，$25B 加速回购正在执行，即使增速不加速，EPS 增长也有机械性支撑；13x Non-GAAP Forward PE 是历史最低区间。
3. **cRPO +16% 先行指标领先营收**（事实）：cRPO 增速持续领先当期营收增速约 4-5pp，按历史规律，FY2027 Q2-Q3 营收应有加速可能。

### C. 必盯指标

- **cRPO 增速**（季度，>16% 为强，<12% 为弱）
- **Agentforce ARR 季度环比增速**（目标：FY2027 末达 $1.5B+）
- **分析师 FY2027E EPS 是否持续上修**（当前 $13.20，若升至 $14+ 则信号强烈）
- **FCF Margin 是否维持 >30%**（管理层长期目标）

### D. 核心雷点（按分级排序）

#### 高风险型

1. **Agentforce 不能货币化（增速高但转化慢）**：当前 29,000 笔交易中，实际 ARR 对客均值约 $27,000/笔，非常小——若大客户不愿意为 Agent 付溢价而仅用免费 tier，$800M ARR 可能触及天花板。**触发逻辑证伪条件**：FY2027 Q2 Agentforce ARR 环比停止加速。
2. **Microsoft 生态侵蚀加速**：60% Fortune 500 已使用 Copilot，若 Copilot Studio AI Agent 能力超预期，"Microsoft-only" 企业减少 Salesforce 实施需求，cRPO 增速下行。
3. **商誉减值风险**：~$50B 商誉（主要是 Slack、Tableau、MuleSoft），若 Slack 增速不及预期（Slack 当前集成进 Platform 分部，无独立披露），叠加 Informatica 整合风险，减值风险非零。

#### 观察型

4. **SBC 长期稀释**：$3.51B/年 SBC，虽被回购大幅中和，但若回购节奏放缓（如 Informatica 现金消耗），稀释将再次显现
5. **AI叙事期权估值成分高但收入占比仍低**：Agentforce $800M ARR / 总收入 $41.5B = 1.9%，Agentforce 叙事比实际贡献领先 1-2 年，若市场耐心耗尽，叙事溢价可能提前回撤
