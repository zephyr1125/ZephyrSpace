# 亚翔集成（603929.SH）

> **A股稀缺的半导体洁净室工程龙头。**  
> 台湾母公司亚翔工程（YHSC）直接控股53.99%，持台积电/中芯/长鑫等头部客户资质，是中国大陆最大的台资洁净室集成EPC商。

## 公司简介

亚翔集成（亚翔工程集成（苏州）股份有限公司）于2016年12月30日登陆上交所主板，是**台湾亚翔工程股份有限公司（YHSC）**的A股上市子公司。公司专注洁净室集成工程EPC总包，服务半导体晶圆厂、面板厂、光伏工厂和医药工厂，是国内少有的拥有台积电标准技术体系的工程商。

- 上市日期：2016-12-30
- 所属行业：装修装饰 / 专业工程
- 总股本：约2.133亿股，流通股约0.965亿股（46%）

## 产业链位置

```
下游资本开支周期（半导体/光伏）
        ↓
  设备采购 & 工厂建设规划
        ↓
亚翔集成（洁净室EPC总包）← 竞品：汉唐集成(688137)
  ├─ 洁净室系统设计
  ├─ 空调暖通(HVAC)
  ├─ 气体/超纯水管路(UPW)
  ├─ 洁净室装修（地板/吊顶）
  └─ 配套机电
        ↓
  竣工交付 → 工厂投产
        ↓
 半导体晶圆代工 / 面板 / 光伏
```

## 核心业务

1. **洁净室集成系统**（主营，约80%以上）：EPC总包，Class 1/10/100洁净度工程
2. **机电配套工程**：电气、给排水、配套设施
3. **运维服务**（少量）：竣工后运维支持

典型项目工期6-24个月，单项目合同金额数千万至数亿元。收入按**完工百分比法**确认，下半年（Q3/Q4）是主要交付期，季度分布极不均匀。
