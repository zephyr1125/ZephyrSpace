# 亚翔集成（603929.SH）

> **A股稀缺的半导体洁净室工程龙头。**  
> 台湾母公司亚翔工程（YHSC）直接控股53.99%，持台积电/中芯/长鑫等头部客户资质，是中国大陆最大的台资洁净室集成EPC商。

## 公司简介

亚翔集成（亚翔工程集成（苏州）股份有限公司）于2016年12月30日登陆上交所主板，是**台湾亚翔工程股份有限公司（YHSC）**的A股上市子公司。公司专注洁净室集成工程EPC总包，服务半导体晶圆厂、面板厂、光伏工厂和医药工厂，是国内少有的拥有台积电标准技术体系的工程商。

- 上市日期：2016-12-30
- 所属行业：装修装饰 / 专业工程
- 总股本：约2.133亿股，流通股约0.965亿股（46%）

## 产业链位置

```
下游资本开支周期（半导体/光伏）
        ↓
  设备采购 & 工厂建设规划

亚翔集成（洁净室EPC总包）← 竞品：汉唐集成(688137)

```

## 核心业务

1. **洁净室集成系统**（主营，约80%以上）：EPC总包，Class 1/10/100洁净度工程
2. **机电配套工程**：电气、给排水、配套设施
3. **运维服务**（少量）：竣工后运维支持

典型项目工期6-24个月，单项目合同金额数千万至数亿元。收入按**完工百分比法**确认，下半年（Q3/Q4）是主要交付期，季度分布极不均匀。

亚翔集成**属于强周期型公司（工程类CapEx驱动）**，其收入直接跟踪下游半导体/光伏行业资本开支。

```
2022-2023：半导体去库存，扩产放缓
2023Q4-2024：半导体库存出清，新一轮扩产启动 ← 亚翔营收爆发+68%
2024H2-2025：部分光伏项目收尾，半导体订单结构优化 ← 营收略降但利润提升
2026Q1：营收+35%、净利+202%  ← 新一轮上升周期信号确立
2026-2027E：AI/HBM/国产替代驱动新一轮晶圆厂建设潮（潜在超预期）
```

**光伏建设周期（次要驱动）：**
- 2022-2024年光伏扩产高峰带来工厂建设需求，亚翔受益
- 2025年起光伏产能过剩，扩产大幅放缓，对亚翔营收的拖累已基本结束
- 未来若TOPCon/钙钛矿工厂有建设需求，可能再度贡献

**当前周期位置：🔵 复苏中期**
- 上升信号：2026Q1营收+35%、利润+202%已确认
- 潜在动力：国产替代加速（长鑫二期、长江存储扩建、中芯北京/上海新厂）

**周期股的操作原则**：在周期上升期前段/中段布局，高峰期（营收增速开始放缓、合同负债开始下降）减仓，不持有到周期顶部。
