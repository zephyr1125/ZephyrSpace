# 佰维存储（688525.SH）

公司代码：`688525.SH`｜深圳佰维存储科技股份有限公司｜科创板

## 在相关指数 / ETF / 行业中的角色

- **角色定位**：周期价值（存储周期弹性仓）
- **所属指数**：931912 中证半导体产业指数
- **主要意义**：国内存储模组封测龙头，纯粹受益于 NAND Flash / DRAM 价格周期
- **实战看点**：盯季度毛利率变化（反映存储均价走势），以及存货/应收账款比率

## 公司简介

深圳佰维存储科技股份有限公司，2022年12月在科创板上市，是国内头部存储模组厂商，核心业务为**嵌入式存储**（eMMC/UFS/eMCP，面向手机及IoT）和**PC存储**（SSD/DRAM模组）。公司本身不制造存储颗粒，属于下游模组封测和销售环节，盈利能力高度依赖 NAND Flash 原料价格周期。2025年下半年~2026年Q1正处于本轮存储价格强复苏周期的高景气阶段。

⚠️ 这是一个典型的**周期股陷阱**！
- 2023–2025年上半年 PE 极高（100x–585x）是因为盈利极差/亏损
- 现在 PE 极低（31.8x）是因为 **Q1 2026 盈利突然爆炸**
- 应切换为 **前向 PE（Forward PE）** 判断，而非依赖 TTM PE 分位

### Forward PE 分析

| 场景 | 假设 | Forward PE |
|---|---|---|
| 乐观（Q1 2026 可持续）| EPS 年化 = 6.21×4 = 24.84元 | 266.5 / 24.84 = **10.7x** |
| 中性（Q2-Q4 回落 50%）| EPS 年化 ≈ 12元 | 266.5 / 12 = **22.2x** |
| 悲观（H2 存储价格回落）| EPS 年化 ≈ 5元 | 266.5 / 5 = **53.3x** |

- **隐含增速**：若市场给予 25x PE，则隐含 EPS ≈ 10.7元，对应全年归母净利 ≈ 50亿，是 FY2025 的 5.9 倍
- **PEG（基于未来预期增速 50%）**：31.8 / 50 = **0.64**（PEG < 1 理论上合理，但增速可持续性存疑）
