# 汇川技术

公司代码：`300124.SZ`｜创业板｜深圳

## 公司简介

汇川技术成立于 2003 年，总部深圳，2010 年登陆创业板，是中国工业自动化控制系统的龙头企业。主营三大业务：**通用自动化**（变频器、伺服驱动器、PLC）、**新能源汽车电驱系统**（电机控制器、电驱总成）、**电梯一体化控制系统**，近年积极布局工业机器人控制器与人形机器人关节伺服。

被市场称为"工业自动化茅台"，核心逻辑是持续从西门子、三菱、安川等外资品牌手中获取国产替代份额。公司属性上是**平台型成长股**：通用自动化是稳定的高毛利利润引擎，新能源电驱是快速扩张的低毛利规模业务，机器人控制器是新增长引擎。

## 在相关指数 / ETF / 行业中的角色

- **角色定位**：`做大脑`——汇川提供运动控制核心（变频器 / 伺服 / PLC / 机器人控制器），是制造业自动化的神经中枢
- **主要意义**：中证机器人指数（931571）第 2 大权重股（9.62%），是 A 股机器人赛道能否兑现逻辑的核心验证者
- **实战看点**：每季度毛利率走势 + 单季利润增速（2025Q3 仅 +4%，警示信号）；机器人控制器年报披露情况

## 产业链位置

汇川技术在工业自动化产业链中处于**中游核心控制层**：

```
上游：IGBT / MCU / DSP 芯片、磁性材料、精密元器件
         ↓
【汇川技术】运动控制系统：变频器 → 伺服驱动器+电机 → PLC → 工业机器人控制器
         ↓
下游：机械装备 OEM → 工厂自动化（汽车 / 3C / 纺织 / 包装） → 终端工厂
```

**新能源汽车电驱链：**

```
上游：功率芯片、铜材、硅钢片
         ↓
【汇川技术】电机控制器（MCU）+ 电驱总成（电机 + 减速器 + 控制器）
         ↓
下游：理想、小鹏、奇瑞、吉利等主机厂
```

**人形机器人产业链：**

```
【汇川技术】伺服驱动 + 运动控制器
→ 关节模组（与减速器、传感器厂商组合）
→ 优必选、宇树、小米机器人等整机厂
```

竞争地位（推断，基于行业报告及公开信息）：
- 低压通用变频器国内份额约 25-30%，第一；已大幅超越十年前水平
- 伺服驱动器国内份额约 20%，第一，正在追赶并超越安川
- 新能源汽车电驱国内前 5，主要竞争者为精进电动、麦格纳、大众自有产线

## 研究观察

### 增长质量（模块 3）

- **收入**：2025 前三季度同比 +24.7%，增速健康。但毛利率从 2024Q1 34.6% → 2025Q3 29.3%，连续 5 个季度下行（**事实**，来源 tushare `fina_indicator`）
- **净利润结构**：2024 全年净利润比 2023 年下滑 9.5%（47.4→42.9 亿），2025 年恢复增长，但 2025Q3 单季增速仅 4%，营收增长 21% 而利润仅增 4%，剪刀差扩大（**高风险事实**）
- **扣非**：未单独核实，汇川利润结构较纯粹，**推断**扣非与净利润差距不大，**待核实**

### 现金流与营运资金（模块 4）

- 2024 全年经营现金流 72 亿，质量优秀；2025Q1-Q3 自由现金流 -7.4 亿，主因资本开支加大（新能源汽车产能投入）（**事实**）
- 应收账款 2025Q3 末 101.9 亿，占前三季度营收 32.2%，绝对额大（**高风险观察项**）
- 存货 85.1 亿（2025Q3），同比有所增长，符合营收扩张节奏，暂无异常堆积

### 资本结构（模块 5）

- 有息负债合计约 37 亿，占净资产仅 10.8%，**财务结构极度稳健**（事实）
- 货币资金 69.1 亿，扣除有息负债后净现金约 +32 亿，**无再融资压力**（事实）
- 资产负债率 46.8%，其中大部分为经营性负债（预付款、应付账款等），非有息杠杆

## 买入逻辑

1. **国产替代长期逻辑**：中国制造业自动化外资品牌（西门子、安川、三菱）历史份额约 50-60%，汇川持续替代。**验证方式**：每季度通用自动化板块收入同比增速是否持续 >20%，毛利率是否维持 >30%

2. **人形机器人控制器产业化加速**：2025-2026 年国内人形机器人整机企业进入量产阶段，汇川伺服驱动 + 运动控制器是核心零部件。**验证方式**：2025 年报能否单独披露机器人相关收入；主要整机厂的采购公告

3. **财务结构干净，无杠杆隐患**：有息负债 37 亿 vs 货币资金 69 亿，净现金状态，无定增摊薄压力。**验证方式**：持续跟踪融资公告（无融资 = 正面）

4. **新能源汽车电驱成本优化**：随规模效应和产品迭代，电驱毛利率有望从当前 15-20% 逐步回升，带动整体毛利率触底反弹。**验证方式**：2025Q4 及 2025 年报毛利率环比是否止跌企稳
