# 商业发射

## 主题说明

商业发射是整个太空经济的"入口税"——所有卫星、载荷、航天器都必须先上天。可复用技术正在重塑发射成本结构，是当前航天产业链中技术迭代最快、竞争格局变化最明显的领域。

**投资视角：** 这是寻找长期持有标的的第一优先领域。

## 核心公司

- [[01-公司/SpaceX]] — 锚点公司，可复用发射的标杆，暂不可直接投资
- [[01-公司/Rocket Lab]] — 最具代表性的上市发射公司，Electron 已验证 + Neutron 待验证
- [[01-公司/Firefly Aerospace]] — 小型火箭 Alpha + 月球着陆器 Blue Ghost，新上市

## 传统发射参与者

- [[01-公司/Boeing]] — SLS 核心级 + ULA 合资方，成本加成模式的代表
- [[01-公司/Lockheed Martin]] — ULA 合资方 + Orion 飞船，政府发射市场
- [[01-公司/Northrop Grumman]] — 固体助推器供应 + Cygnus 货运 + 与 Firefly 合作 MLV
- [[01-公司/RTX Corp]] — RL10 上面级发动机，发射推进供应链咽喉
