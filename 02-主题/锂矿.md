# 锂矿主题

## 主题说明

锂是电动汽车、储能电池的核心金属，下游绑定新能源大趋势。但**锂矿是典型商品价格周期股**，投资节奏必须跟着"碳酸锂/氢氧化锂现货价格周期"走，而非简单看新能源景气度。

**当前周期位置（截至2026-04）：出清期末段 → 等待反转信号**

> 碳酸锂现货价从2023年初峰值约60万元/吨跌至当前约7万元/吨，跌幅超88%。从行业Capex收缩、弱势企业减产、价格企稳迹象来看，出清已接近尾声——但股价已率先反映这一预期，**当前PB均处于5年高分位（88-98%），市场已提前定价复苏**，当前并非理想买入时机。

| 永兴材料 | 3.11 | 1.31 | 3.54 | **93.3%** | 接近5年顶部 |
| 融捷股份 | 6.05 | 2.08 | 6.22 | **96.7%** | 接近5年顶部 |

**为何PB在业绩底部却在高位？**

这是锂矿周期特有的"账面价值侵蚀效应"：
1. 2021-2022锂价暴涨 → 巨额净利润 → 净资产（分母）大幅积累
2. 2023-2025锂价暴跌 → 净利润转负 → 净资产快速缩水
3. 净资产缩水 > 股价跌幅 → PB比率反而升至5年高点
4. 加之市场"抢跑复苏" → 股价反弹 + 净资产仍低 → PB更高

**结论：市场已将2026-2027年锂价复苏预期大部分定价进去**。当前不是"周期底部PB便宜"买入时机，而是"等待盈利验证"的观察期。

## 板块风险提示

1. **碳酸锂价格复苏慢于预期**：当前约7万元/吨，即使反弹至10-12万元/吨，多数公司仍难盈利
2. **产能过剩消化周期**：国内锂矿新增产能仍在释放，供给侧完全出清可能需2-3年
3. **需求侧放缓**：EV渗透率高基数效应，储能需求增量能否对冲EV增速下行存疑
4. **股价已提前反映复苏**：下行修正风险较大，一旦复苏节奏低于预期将显著回调
5. **PB账面价值陷阱**：继续亏损将进一步侵蚀净资产，推高PB，"账面便宜"假象可能持续
