# 工业软件 / 自主可控主题

## 主题定义

本主题聚焦 A 股软件与信息技术服务行业中**国产替代逻辑最强的细分领域**，涵盖：
- **工业软件**：CAD/CAE/ERP/MES 等工业控制和制造流程软件
- **网络安全/信创安全**：SD-WAN、防火墙、NGFW、安全中台（XDR）
- **国产芯片/算力**：国产 CPU、GPGPU（DCU），数据中心算力替代
- **通用办公软件**：WPS、金融数据平台的国产化替代
- **服务器/超算**：国产服务器组装、高性能计算解决方案

**无直接对应指数**：测试了 930608（中证软件服务）、930882（中证软件龙头）等，均不符合主题（成分股为建筑、基建公司），因此采用**自建精选股票池（19 家）**进行粗筛。

## 赛道真相：A股"工业软件"的现实

> **重要认知**：A 股中真正的"工业软件"公司（广联达、中控技术、中望软件等）ROE 普遍低于 12%，大多处于亏损边缘；信创安全公司（奇安信、启明星辰）也大多亏损。财务质量真正达标的反而是**金融数据软件（同花顺）、通用办公（金山）、国产 CPU（海光）、网络安全头部（深信服）、服务器（浪潮、曙光）**。

这揭示了一个重要分层：
- **叙事层**（工业软件国产替代）≠ **财务层**（真正盈利的公司）
- 投资时要区分"最符合主题叙事"和"最有盈利质量的公司"

## 粗筛参数与结果汇总

**行业类型**：成长科技  
**参数**：市值下限 ≥40 亿、ROE ≥12%、PE >0、营收同比 ≥-10%

| 指标 | 门槛 | 备注 |
|---|---|---|
| 市值 | ≥40 亿 | 全部通过 |
| ROE | ≥12%（偏离≤20% 标注⚠️） | 主要筛选门槛 |
| PE | >0（亏损排除） | 不作上限 |
| 营收同比 | ≥-10% | 收缩期的工业软件一律排除 |

**结果**：19 家 → **6 只通过**（其中 3 家已有完整 PreBuy）

## 板块风险提示

1. **A股工业软件真空**：真正的工业软件（广联达、中控、中望）财务质量普遍不达标，叙事与盈利脱节
2. **美国制裁传导**：海光信息的AMD授权问题如扩大，可能传导至整个信创半导体板块信心
3. **政府财政节奏风险**：深信服、中科曙光等以政企采购为主，受财政支出节奏约束，季度波动大
4. **AI叙事泡沫**：同花顺、海光均存在估值已充分甚至超额反映AI叙事的风险
