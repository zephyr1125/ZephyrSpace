# 930713 中证人工智能主题指数

## 指数简介

中证人工智能主题指数（930713，简称"CS人工智"）由中证指数有限公司编制，旨在反映A股市场中与人工智能基础资源、技术研发及应用支持密切相关的上市公司整体表现。指数于2012年6月29日发布，以当日1000点为基点，采用**自由流通市值加权**，成分股数量为50只。

**编制规则要点**：
- 样本空间为中证全指成份股（全部A股）；
- 剔除近一年日均成交金额排名后20%的低流动性标的；
- 主题筛选：涉及AI基础资源（大数据、云计算、智能芯片等）、AI技术（机器学习、语音视觉识别等）、AI应用支持的公司纳入候选池；
- 按近一年日均总市值由高至低排序，选取前50名；
- 单一成分股权重上限10%，前五大成分股合计不超40%；
- 每半年调整一次（6月、12月第二周五后的首个交易日）。

## 指数结构分析

- **算力基础设施链（建骨架/铺血管）**：中际旭创、新易盛（光模块）、中科曙光、浪潮信息（算力服务器）——合计权重超26%，是指数弹性最大的方向，与全球数据中心资本开支直接挂钩。
- **半导体/芯片（做大脑/建骨架）**：寒武纪（AI推理芯片）、澜起科技（内存接口芯片）、豪威集团（CIS图像传感器）——合计约19%，国产替代和AI算力升级双重驱动。
- **AI应用层（做大脑/吃升级）**：科大讯飞（语音大模型应用）、金山办公（AI办公）、海康威视（AIoT安防）——合计约11%，落地确定性相对较强但弹性不如硬件链。

**综合判断**：930713是一个典型的"AI算力硬件优先 + 国产替代逻辑"的主题指数。前两大成分股（中际旭创+新易盛）合计权重约20%，指数表现与光模块景气度高度相关。该指数成长属性强，Beta大，不适合作稳健底仓，适合在AI景气周期上行时配置，下行时需警惕回撤幅度。

## 投资看点与风险

1. **主线逻辑清晰**：全球AI算力军备竞赛持续，数据中心大规模扩建带动光模块、芯片、服务器需求链式增长，指数前四大权重股直接受益。
2. **国产替代加速**：寒武纪、澜起科技、豪威集团等代表AI芯片国产化进程，政策端持续支持，有中长期结构性机会。
3. **周期属性不可忽视**：光模块（中际旭创/新易盛）属于典型周期成长，800G向1.6T切换期间存在价格下行压力，毛利率有可能阶段性承压，需配合景气度判断仓位。
4. **估值分化明显**：寒武纪PE高达220x（2025E），科大讯飞利润承压，指数整体市盈率不低，需动态跟踪各权重股业绩兑现情况。
5. **主要风险**：北美云厂商资本开支放缓、贸易摩擦升级（关税/芯片出口管制）、国内AI应用落地速度不及预期、光模块产能过剩导致价格战。
